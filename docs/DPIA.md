# Data Protection Impact Assessment (DPIA)

**Last updated:** 2026-02-04

## 1) Description of Processing
- System: Kurs Bot
- Purpose: coaching/lesson delivery and personalization
- Data types: identifiers, contact details, message content, derived memories
- Data subjects: end users

## 2) Necessity & Proportionality
- Describe why each data category is necessary.
- Explain minimization measures and defaults.

## 3) Risks to Data Subjects
- Unauthorized access to message content
- Unintended profiling or inference
- Data leakage through logs

## 4) Mitigations
- Authentication and access control
- Logging redaction
- Retention limits and deletion
- Encryption in transit and at rest

## 5) Residual Risk
- Evaluate residual risk and acceptance decision.

## 6) Approval
- DPO/Owner sign-off and date
